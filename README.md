# Homelab

A production-like Kubernetes platform built for learning, experimenting, and demonstrating modern DevOps practices.

> ⚠️ This project is continuously evolving as I learn new technologies and improve the platform.

---

# Goals

This homelab is designed to simulate a real production environment and improve hands-on experience with Kubernetes, Linux, GitOps, observability, databases and infrastructure automation.

The project focuses on:

- Kubernetes administration
- GitOps workflows
- Infrastructure automation
- Monitoring and observability
- Networking
- CI/CD
- High availability concepts

---

# Technology Stack

| Category | Technologies |
|----------|--------------|
| Container Runtime | Docker |
| Kubernetes | k3d |
| Package Management | Helm |
| GitOps | ArgoCD |
| Ingress | Traefik |
| Monitoring | Prometheus, Grafana |
| Database | PostgreSQL |
| Messaging | Apache Kafka |
| Operating System | Linux |
| Version Control | Git |
